Account

  • Create an account to manage your passwords through.
  • Users can DELETE their accounts at any time. However all passwords belonging to the user will be deleted as well.

Passwords

  • *** IMPORTANT: *** Using the same service name (case sensitive) will generate the same password. If multiple passwords are necessary for the same service, consider adding numbers or symbols to the name to generate different passwords.
  • Save your passwords by entering a service to create a password for and any notes about it that you wish to make if desired.
  • Passwords are generated randomly and will be saved with the individual service inputted by the user.
  • Passwords can be viewed by clicking on the card with the respective service.
  • Since passwords are randomly generated, users cannot change passwords for a service. New passwords can be generated by making another service, keeping in mind that a different service name needs to be used to generate a different password.
